História e Evolução dos computadores

A primeira geração de computadores, tem como exemplo o ENIAC * (1946). Foi o primeiro computador digital e eletrônico programável do mundo. E o enorme Colossus* fazem parte dos primeiros computadores.

A segunda geração com transistores, substituindo as válvulas de 1959 até 1965.
Na terceira geração funcionavam por circuitos integrados, substituindo os transistores de 1965 até 1970.

A partir de 1971 temos a quarta geração que dura até hoje, foi nela que os computadores evoluiram até o que conhecemos hoje.

há quem diga que estamos vivendo a quinta geração de computadores que seria marcada pelas criptomoedas, metaverso e IA.

O que são Clients e Servers?

O modelo cliente-servidor (em inglês client/server model), em computação, é uma estrutura de aplicação distribuída que distribui as tarefas e cargas de trabalho entre os fornecedores de um recurso ou serviço, designados como servidores/servers, e os requerentes dos serviços, designados como clientes/clients.

CLIENTS: É o cliente quem sempre inicia a comunicação com o servidor. O cliente mais conhecido é o navegador. Quando acessamos um site pelo navegador, uma requisição é disparada pela internet para o computador no qual esse site está hospedado.,

SERVERS: O papel do servidor é receber uma requisição e devolver uma resposta para o cliente.

Como é uma estrutura básica de um HTML?

A estrutura básica* da Linguagem de Marcação de Hipertexto HTML, se consiste em 3 tags principais, sendo elas:

  1. HTML - Na estrutura do documento, antes de começar a colocar o conteúdo, inserimos uma tag html, define o documento;
  2. HEAD - Define o cabeçalho da página, que contém informações para o navegador e que não aparecerá para o usuário;
  3. BODY - Define o corpo de um documento html, que é exibido pelo navegador em sua janela, ou seja, todo o conteúdo visível do site.

Entendendo a diferença entre listas ordenadas e não ordenadas

Em HTML podemos fazer dois tipos de listas listas ordenada e não ordenada.

  • Ordenadas - Utiliza a tag ol
  • Não ordenadas - Utiliza a tag ul

As listas ordenads, seguem uma ordem e são mostradas em sequência numerada para cada item da lista, por exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

As listas não ordenadas são mostradas com pontos ao lado dos itens, por exemplo:

  • Item 1
  • Item 2
  • Item 3
